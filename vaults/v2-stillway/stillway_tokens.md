## SECTION 3: HYPERFRAMES PIPELINE

```
INPUT: Tier 2 request → "generate HyperFrames tokens for [protocol] at [context]"
          |
          v
  1. Fetch protocol from BREATH_SYNC_PROTOCOLS
  2. Map to base tokens via BREATH_TO_TOKEN_MAP
  3. Apply context modifiers (onboarding | negotiation | recovery)
  4. Generate full CSS custom properties block
  5. Output: design_tokens.json + tokens.css
  6. Store in Vault 2 with session_id and timestamp
```
